Overview

Project Overview

프로젝트 개요

My Health Coach is a comprehensive chronic disease management platform built with Flutter. It consolidates health monitoring across five clinical domains — diabetes, hypertension, obesity, cancer, and general wellness — into a single, AI-assisted mobile experience.

당뇨·고혈압·비만·암·웰니스 5개 만성질환 영역을 하나의 앱에서 통합 관리하는 Flutter 기반 헬스케어 플랫폼입니다.

Each disease module provides a 5-tab dashboard (AI Assessment · AI Counseling · Record History · Lab Results · Management Status), enabling continuous health tracking. The backend is built on Spring Boot 3.5 with KR Core FHIR R4, ensuring full compliance with Korean healthcare interoperability standards.

각 질환 모듈은 종합평가·AI 검진사항·기록이력·검사결과·관리현황 5개 탭으로 구성되며, Spring Boot + KR Core FHIR R4 백엔드로 한국 의료 표준을 준수합니다.

A standout feature is the Wellness AI Agent Control Center, which orchestrates multiple Gemini-powered agents — Diet Analysis, Blood Sugar Analysis, Blood Pressure Analysis, and Comprehensive Daily Report — through a real-time Agent Ops pipeline with live execution logs.

웰니스 AI Agent 컨트롤 센터는 식단·혈당·혈압 분석과 일일 종합 리포트를 Google Gemini 기반 멀티 에이전트로 실시간 실행하며, Agent Ops 로그를 화면에서 직접 확인할 수 있습니다.

Key Screens

App Screens

주요 화면

당뇨 집중 관리 · 5-Tab Dashboard
The diabetes module delivers a 5-tab dashboard: Comprehensive Assessment (blood glucose trend chart with goal zones), AI Counseling (priority alerts, lifestyle recommendations), Record History (CGM / manual entries), Lab Results (HbA1c, eGFR, lipid timeline), and Management Status (medication history).
종합평가(혈당 추이·목표 구간 그래프)·AI 검진사항(우선 경고·생활습관 조언)·기록이력(CGM/직접입력)·검사결과(HbA1c·eGFR·지질 타임라인)·관리현황(복약이력) 5개 탭으로 구성됩니다.
고혈압 집중 관리 · 5-Tab Dashboard
Hypertension management mirrors the diabetes module's 5-tab structure with domain-specific content: systolic/diastolic trend graphs, AI priority warnings (salt restriction, aerobic exercise), blood pressure record timeline, and complication screening lab results (lipid panel, electrolytes).
수축기·이완기 혈압 추이 그래프, 우선 경고(염분 제한·유산소 운동 권고), 혈압 기록 타임라인, 합병증 스크리닝 검사(지질·전해질)가 동일한 5탭 구조로 제공됩니다.
암 환자 케어 · Cancer Care Module
Cancer care hub: pain logging (multi-site, intensity scale, type tags), upcoming appointment management, AI meal planner, medication schedule, and blood test results — all tailored for oncology patients.
통증 기록(복수 부위·강도·유형), 진료 일정, AI 식단 추천, 복약 스케줄, 혈액검사 결과를 암 환자 특화 UI로 통합 제공합니다.
복약 스케줄 · AI 약물 도우미
Medication tracker with calendar view and AI Drug Assistant: camera-based OCR identifies prescriptions, AI performs safety analysis (drug interaction check, conflict detection), and generates a structured counseling report.
캘린더형 복약 스케줄 + AI 약물 도우미: 카메라로 처방전을 촬영하면 AI가 약물 인식·안전성 검토·상호작용 경고·복약 지도 리포트를 자동 생성합니다.
혈액검사 AI 리포트 · Lab AI Report
4-tab blood test report: Summary (patient profile, current status), Detailed Results (CBC, tumor markers with trend charts), AI Report (positive findings, counseling needs, lifestyle guide), and Treatment History (medication timeline).
종합요약(환자 정보·현 상태)·상세결과(CBC·종양표지자 추이)·AI 리포트(긍정소견·주의관찰·생활습관 가이드)·치료이력(복약 타임라인) 4탭으로 구성됩니다.
웰니스 AI Agent 컨트롤 센터
Multi-agent orchestration center: select from Diet Analysis, Blood Sugar Analysis, Blood Pressure Analysis, or Comprehensive Daily Report agents. Displays real-time execution state (LLM engine, patient scope, execution ID) and live Agent Ops logs.
식단·혈당·혈압 분석 및 일일 종합 리포트 에이전트를 선택 실행. 현재 LLM 모델·환자 범위·실행 ID를 표시하고 실시간 Agent Ops 로그를 스트리밍합니다.
AI 건강교실 · AI Health Education
Personalized health education powered by AI: browse categorized health topics (cancer basics, early symptoms, treatment options, nutrition) and open an AI chat interface for 1-on-1 Q&A tailored to the user's health profile and condition history.
암 기초·초기증상·치료법·영양 등 건강 토픽을 카테고리별로 탐색하고, 내 건강 상태에 맞춘 AI 1:1 채팅으로 심화 질문을 할 수 있습니다.
