Overview

Project Overview

프로젝트 개요

Nursing students practice in a real hospital EMR environment — covering OCS, ENR, Lab, and PACS — using GUIDED or EVALUATION simulation modes.

간호대학생들이 실제 병원 EMR과 동일한 환경(OCS, ENR, Lab, PACS)에서 GUIDED 또는 EVALUATION 모드로 실습합니다.

Upon completion, 6 AI Agents automatically evaluate each student's performance. The full pipeline — LLM call, JSON validation, professor review, and result publication — is fully automated.

실습 완료 후 6개 AI Agent가 자동으로 평가합니다. LLM 호출 → JSON 검증 → 교수 검토 → 결과 공개 전 과정이 자동화되어 있습니다.

The system has processed 40,000+ evaluation log entries and reduced professor evaluation time by 90%.

누적 40,000+건의 평가 로그를 처리했으며, 교수님의 평가 시간을 90% 단축했습니다.

System Architecture

시스템 아키텍처

Layered AI Pipeline: EvaluationController → NursingEvaluation OrchestrationService → Execution Layer (@Transactional) → AI Agent Components → Google Vertex AI (Gemini 2.0 Flash). All AI responses are validated via JSON Schema before being saved to the database.
EvaluationController → OrchestrationService → 실행 레이어(@Transactional) → AI Agent → Google Vertex AI (Gemini 2.0 Flash) 순으로 처리됩니다. 모든 AI 응답은 JSON Schema 검증 후 DB에 저장됩니다.

Features

Key Screens

주요 화면

간호기록 (ENR) · GUIDED MODE
Students practice in a real hospital EMR environment. Records Initial Assessment, Vital Signs, I/O, and Nursing Notes in real time with step-by-step progress tracking.
실제 병원 EMR과 동일한 환경에서 간호 초기사정, 활력징후, 섭취/배설, 간호노트를 실시간 기록합니다. 우측 패널에서 실습 진행률을 단계별로 확인할 수 있습니다.
검사결과 (Lab)
CBC, biochemistry results with trend charts. Critical values highlighted, nursing action reports auto-generated.
CBC, 생화학 결과를 추이 차트와 함께 제공. 위험 수치 강조 및 간호 조치 보고서 자동 생성.
영상판독 (PACS)
X-Ray, CT, MRI images with diagnostic reports. Findings + Impression sections simulate real clinical reading.
X-Ray, CT, MRI 영상과 판독 보고서 제공. Findings/Impression으로 실제 임상 판독 시뮬레이션.
AI 평가 실행 · Vertex AI (Gemini 2.0 Flash)
Full AI prompt, patient context, and JSON response are logged transparently. Each agent's score and detailed feedback visible in real time.
AI 프롬프트, 환자 컨텍스트, JSON 응답이 모두 투명하게 기록되어 교수님이 검증할 수 있습니다. 각 Agent의 점수와 상세 피드백이 실시간으로 표시됩니다.
교수용 평가 관리 대시보드 · Professor View
Professors review AI scores with category-level feedback (correct / caution / needs improvement). Final scores approved or adjusted with one click.
항목별 상세 피드백(잘한 점/개선할 점/주의사항)을 검토하고 최종 점수를 확정하거나 수정합니다. 원클릭으로 승인 또는 조정이 가능합니다.
